# Online Magazine API - Final Project

рЃћрЃА рЃљрЃарЃўрЃА рЃЉрЃћрЃЦрЃћрЃюрЃЊ рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃўрЃА рЃЎрЃБрЃарЃАрЃўрЃА рЃерЃћрЃЏрЃљрЃ»рЃљрЃЏрЃћрЃЉрЃћрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў. рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃЮрЃюрЃџрЃљрЃўрЃю рЃЪрЃБрЃарЃюрЃљрЃџрЃўрЃА API-рЃА, рЃАрЃљрЃЊрЃљрЃф рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ, рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃАрЃбрЃљрЃбрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃљ.

**рЃљрЃЋрЃбрЃЮрЃарЃў:** рЃЌрЃћрЃЏрЃЮ рЃџрЃЮрЃЏрЃўрЃФрЃћ

## ­ЪЏа рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў

* **Node.js & Express** - рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ
* **MongoDB & Mongoose** - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ
* **JWT (JSON Web Token)** - рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ
* **Bcrypt** - рЃърЃљрЃарЃЮрЃџрЃћрЃЉрЃўрЃА рЃ░рЃћрЃерЃўрЃарЃћрЃЉрЃљ
* **Nodemailer** - рЃћрЃџ.рЃцрЃЮрЃАрЃбрЃўрЃА рЃЊрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
* **Multer** - рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ
* **Joi** - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ

## ­Ъџђ рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ

### 1. рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ
рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ:
```bash
npm install
```

### 2. рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў (.env)
рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `.env` рЃцрЃљрЃўрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў рЃЊрЃљ рЃњрЃљрЃгрЃћрЃарЃћрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў:

```env
PORT=3000
MONGODB_URI=рЃЌрЃЦрЃЋрЃћрЃюрЃў_рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ_рЃЉрЃљрЃќрЃўрЃА_рЃџрЃўрЃюрЃЎрЃў
JWT_SECRET=рЃЌрЃЦрЃЋрЃћрЃюрЃў_рЃАрЃљрЃўрЃЊрЃБрЃЏрЃџрЃЮ_рЃЎрЃЮрЃЊрЃў
JWT_EXPIRES_IN=1h
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=рЃЌрЃЦрЃЋрЃћрЃюрЃў_рЃЏрЃћрЃўрЃџрЃў@gmail.com
SMTP_PASS=рЃЌрЃЦрЃЋрЃћрЃюрЃў_app_password
BASE_URL=http://localhost:3000
```

### 3. рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ
```bash
npm run dev
```

## ­ЪЊА API рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ

### ­Ъћљ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ (Auth)

* **POST** `/api/auth/register`
  * рЃљрЃ«рЃљрЃџрЃў рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ.
  * **рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў:** рЃАрЃљрЃ«рЃћрЃџрЃў (min 3 рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮ), рЃърЃљрЃарЃЮрЃџрЃў (min 8 рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮ).
  
* **POST** `/api/auth/login`
  * рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў рЃерЃћрЃАрЃЋрЃџрЃљ. рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА **JWT рЃбрЃЮрЃЎрЃћрЃюрЃА**.
  
* **GET** `/api/auth/confirm`
  * рЃЏрЃћрЃўрЃџрЃўрЃА рЃЊрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃљ (рЃџрЃўрЃюрЃЎрЃў рЃўрЃњрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ рЃЏрЃћрЃўрЃџрЃќрЃћ).

### ­ЪЊЮ рЃАрЃбрЃљрЃбрЃўрЃћрЃЉрЃў (Articles)
*рЃАрЃбрЃљрЃбрЃўрЃћрЃЉрЃўрЃА рЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЋрЃљрЃЊ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ `Authorization` рЃ░рЃћрЃЊрЃћрЃарЃў: `Bearer YOUR_TOKEN`*

* **GET** `/api/articles` - рЃДрЃЋрЃћрЃџрЃљ рЃАрЃбрЃљрЃбрЃўрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ.
* **POST** `/api/articles` - рЃљрЃ«рЃљрЃџрЃў рЃАрЃбрЃљрЃбрЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ.
* **GET** `/api/articles/:id` - рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃАрЃбрЃљрЃбрЃўрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ ID-рЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ.
* **PUT** `/api/articles/:id` - рЃАрЃбрЃљрЃбрЃўрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ (рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃЋрЃбрЃЮрЃарЃўрЃАрЃЌрЃЋрЃўрЃА).
* **DELETE** `/api/articles/:id` - рЃАрЃбрЃљрЃбрЃўрЃўрЃА рЃгрЃљрЃерЃџрЃљ (рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃЋрЃбрЃЮрЃарЃўрЃАрЃЌрЃЋрЃўрЃА).

### ­ЪЉц рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў (User)

* **GET** `/api/users/me` - рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃърЃарЃЮрЃцрЃўрЃџрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ.
* **PUT** `/api/users/me` - рЃърЃарЃЮрЃцрЃўрЃџрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ (рЃАрЃљрЃ«рЃћрЃџрЃў, рЃљрЃЋрЃљрЃбрЃљрЃарЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ).

## ­ЪЏА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ рЃЊрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ

* **Joi Validation:** рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃЏрЃЮрЃАрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў рЃЏрЃЮрЃгрЃЏрЃЊрЃћрЃЉрЃљ. рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃАрЃљрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏ **3 рЃљрЃАрЃЮ**, рЃ«рЃЮрЃџрЃЮ рЃърЃљрЃарЃЮрЃџрЃў рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏ **8 рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮ**.
* **Protected Routes:** рЃАрЃбрЃљрЃбрЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ, рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃгрЃљрЃерЃџрЃљ рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЊрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃБрЃџ (Confirmed) рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃћрЃЉрЃБрЃџ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА.
* **File Upload:** рЃљрЃЋрЃљрЃбрЃљрЃарЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ `src/uploads/avatars` рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў.
